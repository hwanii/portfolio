CLASSVR
LMS 학생관리
프로그램

CLASSVR의 LMS(학생관리시스템)은
한번의 클릭만으로 모든 기기를 통제 할 수 있는
프로그램 입니다

CLASSVR의 LMS(학생관리시스템)에는
VR,AR,메타버스 콘텐츠가 수천개가 탑재되어
있습니다.

Drag&Drop

원하는 VR,AR,메타버스 콘텐츠를 Drag&Drop으로
신규 재생목록을 쉽게 만들수 있으며
한번의 실행 버튼 클릭 만으로 모든 기기에서 동시에
전송 실행이 되어 누구나 쉽게 사용을 할 수 있습니다

헤드 트랙킹(Head Tracking) 기능

학생 개개인의 헤드셋 시점을 묘사한 작은 아이콘들이
교사의 포탈 화면에 표시 되어 모든 학생들의 시선을
실시간으로 한 화면에서 확인 할 수 있습니다.

CLASS VIEW

VR 헤드셋을 쓰고 있을 때 학생들이 어디를 보고, 무엇을 보고 있는지 파악이 어렵습니다.
Class VR의 혁신적인 헤드 트랙킹 기능은 어떤 학생이 무엇을 보고 있는지 화면을 실시간으로 전송하여
교사의 주도하에 수업의 집중도와 효과를 높이게 합니다.

클라우드에 올려 수업에 활용

CLASSVR의 LMS(학습관리프로그램)는
자체적으로 제작한 콘텐츠를 클라우드로 업로드하여 모든 기기에 연동 시킬 수 있습니다
직접 찍은 360º사진(.jpg), 360º 동영상(.mp4)을 업로드 하여 학생들에게 보여주고
직접 만든 3D모델(.glb)을 클라우드에 올려 수업에 활용하세요
